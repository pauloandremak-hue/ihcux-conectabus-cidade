# ihcux-conectabus-cidade
Mobilidade Inteligente Urbana
Sobre o Projeto
Este projeto consiste em um aplicativo de mobilidade urbana desenvolvido para ajudar usuários a encontrar a melhor rota de transporte de forma rápida e prática.
O aplicativo reúne diferentes meios de transporte em um único sistema, permitindo visualizar trajetos, tempo estimado, custo e condições da rota.

Objetivo

Desenvolver uma solução tecnológica capaz de:
Facilitar a locomoção urbana;
Melhorar a experiência do usuário durante o trajeto;
Auxiliar pessoas com mobilidade reduzida;
Tornar o deslocamento mais acessível e eficiente.
Contexto de Uso

O aplicativo foi pensado para situações do cotidiano, como:

Pessoas correndo para não perder o ônibus ou metrô;
Usuários procurando a rota mais rápida;
Pessoas com dificuldade de locomoção buscando trajetos acessíveis;
Usuários sem conexão momentânea com a internet.

Exemplo:

Um estudante precisa chegar rapidamente à faculdade e utiliza o aplicativo para encontrar a rota mais rápida entre ônibus, bicicleta e carro por aplicativo.

Funcionalidades
Seleção de rotas;
Comparação entre meios de transporte;
Tempo estimado de chegada;
Integração com mapas;
Pagamento via QR Code;
Modo noturno;
Sistema de desafios;
Reconexão automática sem internet.
Decisões de UX

Algumas informações foram destacadas para melhorar a experiência do usuário:

O tempo estimado foi colocado em destaque para facilitar decisões rápidas;
Os meios de transporte utilizam ícones simples e fáceis de identificar;
O QR Code foi centralizado para agilizar o pagamento;
O mapa ocupa grande parte da tela para melhorar a visualização da rota;
O modo noturno reduz o cansaço visual;
Os botões possuem tamanho maior para facilitar o uso em situações de pressa.
Acessibilidade

O aplicativo possui recursos voltados à acessibilidade:

Botões grandes e fáceis de selecionar;
Rotas acessíveis destacadas;
Interface com alto contraste no modo noturno;
Informações organizadas de forma clara;
Funcionamento mesmo com conexão instável.
